Aplikasi catatan dan daftar to-do open source yang menggunakan teknologi Android terbaru.

* Dukungan dash clock (ya, masih berfungsi)
* Notifikasi: atur pengingat pada waktu tertentu, dengan pengulangan opsional pada hari kerja tertentu
* Pemilihan bahasa
* Widget, dengan dukungan layar kunci, dapat diubah ukurannya dan dikonfigurasi
* Lindungi catatan penting dengan kata sandi
* Pindahkan tugas antar daftar yang berbeda
* Seret untuk mengurutkan ulang tugas
* Integrasi dengan widget Agenda Android
* Pencarian
* Gunakan emoji sebagai tag 🏷️
* Tema terang dan gelap
* Cadangkan (impor dan ekspor) catatan ke file JSON
* Sinkronkan dengan file ORG di penyimpanan lokal
* Riwayat catatan dengan semua versi sebelumnya
* Tempat sampah untuk catatan yang dihapus

Untuk meningkatkan aplikasi ini, berdonasilah, atau sekadar berbagi ide,
kunjungi https://github.com/spacecowboy/NotePad
